# Innomine Web

Static GitHub Pages site for `innomine.pw`.

## Structure

```txt
/
  index.html
/contact/
  index.html
/downloads/
  index.html
/assets/
  site.css
  site.js
  profile-main.jpg
  cover-banner.jpg
CNAME
.nojekyll
```

## Routes

- `/` = main page
- `/contact/` = contact page
- `/downloads/` = empty downloads container

## Edit later

In `contact/index.html`, replace:

- `https://simplex.chat/` with your real SimpleX link
- `#` in Federated Network with your federated profile
- `mailto:yourmail@example.com` with your real email
