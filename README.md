# Web de contacto

Página web estática para contactos y redes sociales.

## Archivos

- `index.html`: estructura principal de la página.
- `style.css`: colores, diseño y animaciones.
- `script.js`: interacción básica.

## Qué debes cambiar

En `index.html`, cambia:

- `Tu Nombre o Marca`
- `HN`
- enlaces de WhatsApp, Instagram, Facebook, TikTok y correo
- descripción personal
- texto del footer

## Cómo publicarla

1. Sube estos archivos a tu repositorio de GitHub.
2. Conecta el repositorio con Cloudflare Pages.
3. Agrega tu dominio personalizado en Cloudflare.
